HTML Image

The HTML 'img' tag is used to embed an image in a web page.

In 'img' tag we use 2 attribute to display image. They are src and alt

src-Specifies the path to the image

alt-Specifies an alternate text for the image

For width and height of image we use style atrribute.

We can also use width and height attribute saparately
